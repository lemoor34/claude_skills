## Назначение
Помощь в анализе, оптимизации и масштабировании Meta Ads кампаний для швейцарского рынка (особенно для агентства Swiss SMM Balian и бизнеса Fleissig).

## Триггеры использования
- Кампания не даёт ожидаемый ROI
- Нужна демографическая сегментация по швейцарским кантонам
- Требуется анализ CPA/ROAS по целевым группам
- Нужны рекомендации по языкам и локализации
- Требуется стратегия A/B-тестирования

### Ключевые демографические параметры:

**Age Groups:**
- 18-24: Студенты, первая работа (бюджет средний)
- 25-34: Молодые профессионалы (высокий бюджет, tech-savvy)
- 35-49: Семьи, руководители (высокий бюджет, практичность)
- 50-65: Консервативные, часто высокий доход (традиционные каналы)

**Income Levels:**
- Швейцария — один из самых богатых стран (средняя зарплата 6500-8000 CHF)
- Premium и Mid-range товары имеют хороший ROI
- Low-price — плохой ROI (низкая мотивация)

**Interests по кантонам:**
- Zürich: Технологии, стартапы, premium-услуги
- Bern: Государство, юриспруденция, консервативный бизнес
- Genève: Финансы, роскошь, международный бизнес
- Ticino: Локальная экономика, туризм

## Языки и локализация

### Основные языки:
- **Deutsch (DE)**: ~63% населения (Zürich, Bern, Luzern, Basel, etc.)
- **Français (FR)**: ~23% (Genève, Vaud, Valais, Neuchâtel)
- **Italiano (IT)**: ~8% (Ticino)
- **Romansh (RM)**: ~0.5% (Graubünden) — маркетинг редко используется

### Важные нюансы локализации:

**Швейцарский немецкий vs. стандартный немецкий:**
- Копирайты должны быть на **стандартном немецком** (все поймут)
- Но лучше использовать швейцарские примеры и références (города, традиции)
- Избегать австрийского или баварского диалектов

**Французский:**
- Используется стандартный французский (как в Франции)
- Но упоминать швейцарские особенности (франкоязычные кантоны, стоимость жизни)

## CPA/ROAS анализ по целевым группам

### Типичные CPA по отраслям в Швейцарии:

| Отрасль | Среднее CPA | Хороший CPA | Отличный CPA |
|---------|------------|-----------|-------------|
| E-commerce | 45-80 CHF | 30-45 CHF | <30 CHF |
| SaaS / Services | 80-150 CHF | 50-80 CHF | <50 CHF |
| Lead Generation | 25-50 CHF | 15-25 CHF | <15 CHF |
| Cleaning/Services | 35-70 CHF | 20-35 CHF | <20 CHF |
| Real Estate | 150-300 CHF | 100-150 CHF | <100 CHF |

### Типичный ROAS (Return on Ad Spend):
- **Low ROAS**: <2.0 (убыток или на грани)
- **Normal ROAS**: 2.0-3.5 (стабильно прибыльно)
- **Good ROAS**: 3.5-5.0 (отличный результат)
- **Excellent ROAS**: >5.0 (масштабируем сразу)

### Пример анализа:
```
Кампания: Fleissig Reinigung (Zürich)
Бюджет: 500 CHF/неделя
Leads: 8-10 в неделю
CPA: 50-60 CHF
Целевой CPA: <40 CHF

Вывод: ROAS низкий, нужна оптимизация targeting или креативов.
```

## A/B-тестирование рекомендации

### По языкам:
```
Test 1: Deutsch vs. Français (если целевой регион двуязычный)
Test 2: Standard German vs. Swiss-локализованный немецкий
Test 3: French (Switzerland-focused) vs. France-focused
```

### По демографии:
```
Test 1: 25-34 (молодые профессионалы) vs. 35-49 (семьи)
Test 2: High income (>150K CHF/год) vs. Medium income (80-150K CHF)
Test 3: Urban (Zürich) vs. Suburban (Bern, Luzern)
```

### По креативам:
```
Test 1: Premium (люкс, качество) vs. Practical (результаты, экономия)
Test 2: Images vs. Video vs. Carousel
Test 3: Emotional copy vs. Rational/Facts copy
```

### По landing page:
```
Test 1: Swiss-локализованная LP vs. Generic
Test 2: Language: DE vs. FR
Test 3: CTA: WhatsApp vs. Phone vs. Form
```

## Процесс анализа (пошаговый)

1. **Input**: скриншоты/экспорт из Ads Manager (Campaign stats, Demographics, Geography)
2. **Market check**: сравнение CPA/ROAS с швейцарскими benchmark'ами
3. **Demographic analysis**: кто конвертится лучше всего (возраст, пол, кантон)
4. **Geographic breakdown**: города, кантоны, регионы — где ROI лучше
5. **Language analysis**: если есть многоязычные кампании
6. **Budget allocation**: рекомендации по перераспределению бюджета
7. **A/B test recommendations**: что тестировать дальше
8. **Output**: рекомендации + готовый план действий

## Примеры рекомендаций

### Пример 1: Низкий ROI в Genève
```
Проблема: Кампания в Genève показывает CPA 120 CHF, тогда как в Zürich — 50 CHF.

Гипотеза: Genève — более дорогой и консервативный рынок, нужен другой креатив.

Рекомендация:
- Создать отдельную кампанию для Genève (Premium positioning)
- Аудитория: только High income (>180K CHF/год)
- Креатив: Premium качество, люкс-позиционирование
- CTA: "Бесплатная консультация для VIP-клиентов"
```

### Пример 2: Языковая проблема
```
Проблема: Французский креатив конвертится хуже, чем немецкий (25% lower CTR).

Гипотеза: Копирайт на французском звучит как переводной, не локализованный.

Рекомендация:
- Переписать копирайт специально для франкоязычной Швейцарии
- Использовать швейцарские примеры, города (Genève, Lausanne)
- Избегать французских выражений (которые не используются в Швейцарии)
- Адаптировать CTA (например, "Appel gratuit" вместо "Chat WhatsApp")
```

### Пример 3: Масштабирование с низким бюджетом
```
Проблема: Успешная кампания в Zürich (ROI 4.5:1), но бюджет только 200 CHF/неделя.

Рекомендация:
1. Increase budget в Zürich до 400 CHF (удвоить, если ROI выше 3:1)
2. Запустить идентичную кампанию в Bern и Luzern (похожая демография)
3. Тестировать расширение в другие кантоны (Aargau, Solothurn)
4. Через 2 недели смотреть результаты и масштабировать дальше
```

## Выход

После анализа получишь:
1. **Аналитический отчёт** (текущее состояние, проблемы, возможности)
2. **Бенчмарк-сравнение** (как стоишь относительно рынка)
3. **План оптимизации** (конкретные шаги на 2-4 недели)
4. **A/B-тест рекомендации** (что тестировать, как измерять)
5. **Бюджет-рекомендации** (как перераспределить деньги для максимального ROI)
6. **Готовый чеклист** (действия на неделю/месяц)

## Метрики успеха

- **CPA снижается** на 20-30% за месяц
- **ROAS растёт** с 2.5 до 3.5+ за месяц
- **Масштабирование** без потери ROI (бюджет растёт, но эффективность не падает)
- **Новые клиенты** от оптимизированных кампаний
