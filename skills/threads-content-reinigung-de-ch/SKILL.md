## Назначение
Создание аутентичного, локализованного контента для Threads (и переиспользования на Instagram) по услугам уборки (Reinigung) и ухода за садом (Gartenpflege) для немецкоязычной Швейцарии.

## Триггеры использования
- Нужно создать пост для Threads про услугу уборки
- Нужны идеи сезонного контента (зима, весна, лето, осень)
- Требуется контент с CTA на WhatsApp
- Нужны примеры "день из жизни" профессионала
- Требуется образовательный контент (советы по уборке/саду)

## Типы контента для Threads

### 1. Educational Content (Советы + How-to)
```
Пример темы: "Как убрать снежную соль с деревянной мебели"

Структура:
- Hook: проблема (соль разрушает мебель)
- 3-4 совета (конкретные шаги)
- Product mention (если есть услуга)
- CTA: WhatsApp для консультации

Тон: дружелюбный, экспертный, практичный
```

### 2. Day-in-the-life (Профессиональные будни)
```
Пример: "Мой день начинается в 6:00. Инструменты готовы, план очищен..."

Структура:
- Behind-the-scenes моменты
- Демонстрация профессионализма
- Реальные примеры (фото до/после)
- Humanize the brand (люди, а не корпорация)

Тон: аутентичный, honest, гордость за работу
```

### 3. Before/After (Визуальное преобразование)
```
Пример: фото заброшенного сада → фото ухоженного сада

Структура:
- Сильное визуальное контрастирование
- Короткий текст (2-3 предложения)
- Timeframe (сколько времени заняло)
- CTA: "Хочу такое же!"

Тон: впечатляющий, мотивирующий
```

### 4. Seasonal Tips (Сезонный контент)
```
Пример (весна): "Ваш газон пережил зиму? Вот что нужно сделать в апреле..."

Структура:
- Сезонная актуальность
- Чек-лист действий
- Информация о услугах (если подходит)
- CTA: "Нужна помощь? Напиши в WhatsApp"

Тон: практичный, своевременный, helpful
```

### 5. Industry Insights (Эксперт-контент)
```
Пример: "Почему эко-чистящие средства лучше для паркета"

Структура:
- Исследование или личный опыт
- Объяснение преимуществ
- Статистика (если есть)
- Рекомендация услуги

Тон: авторитетный, информативный
```

### 6. Customer Testimonial (Отзывы)
```
Пример: "Фрау Бауэр из Цюриха: 'Когда она уходит, мой дом сияет...'"

Структура:
- Квота (имя, город)
- Проблема, которую мы решили
- Результат
- CTA: "Хочешь такую же историю успеха?"

Тон: персональный, благодарный
```

## Частота публикации

- **Оптимально**: 3-4 поста в неделю (вторник, четверг, суббота, понедельник)
- **Минимум**: 2 поста в неделю (чтобы алгоритм не забывал)
- **Максимум**: 1 пост в день (в пиковый период, если ряд идей)

## Метрики успеха

Через месяц:
- **Engagement**: 20-30 лайков на пост (стартовый уровень)
- **Reach**: 100-200 уникальных просмотров на пост
- **Conversion**: 1-2 WhatsApp сообщения в неделю от Threads
- **Comments**: 2-5 комментариев на пост (вопросы, запросы)

Через 3 месяца:
- **Followers**: +100-200 подписчиков
- **WhatsApp leads**: 4-8 в неделю (20-30 в месяц)
- **Conversion rate**: 15-25% от leads в заказы
